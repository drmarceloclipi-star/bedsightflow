# BQAgent: Analytics & BigQuery Specialist

The BQAgent manages the data pipeline from Firestore to BigQuery, enabling analytical queries, reporting, and data insights for the LEAN system.

## Specializations

- **Firestore → BigQuery Sync**: Configuring and maintaining the Firebase BigQuery extension and streaming pipeline.
- **Schema Design**: Defining table schemas, partitioning strategies, and clustering for efficient queries.
- **SQL Queries**: Writing analytical queries for operational insights (e.g., bed occupancy trends, discharge rates).
- **Dashboards**: Supporting the creation of Data Studio / Looker dashboards from BigQuery data.
- **Data Quality**: Monitoring data freshness, completeness, and consistency between Firestore and BigQuery.

## Technical Stack

- Google BigQuery (standardize on `southamerica-east1` region)
16: - Firebase Extensions (Firestore → BigQuery)
17: - SQL

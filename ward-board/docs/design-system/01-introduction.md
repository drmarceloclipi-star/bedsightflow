# Design System: Introduction

## Philosophy & Vision

Our design system is heavily inspired by the aesthetic and functional principles of **Claude by Anthropic**. It aims to deliver a **premium, professional, and warmly minimalist** user experience. The system prioritizes extreme clarity, scannability, and reduction of cognitive load, essential for complex and high-stakes applications like Ward Board.

## Core Principles

1. **Warm Minimalism**: UI should never feel cold, sterile, or strictly "tech". We rely on warm off-whites, soft grays, and muted earth tones to make parsing data a more pleasant and humane experience.
2. **Typography as Interface**: Text is the primary interface element. We depend on high-quality, legible typefaces, strict hierarchies, and generous line heights over excessive boxes, cards, or backgrounds.
3. **Restrained Componentry**: Components like buttons and inputs lack heavy borders, harsh drop-shadows, or bright distinct colors unless absolutely necessary for calls to action. We use soft interactive states, smooth hover effects, and subtle contrast changes.
4. **Data Clarity**: Data and information (such as beds, patient names, and alerts) must breathe. Ample whitespace and distinct separation through negative space, rather than literal line dividers, are preferred.
